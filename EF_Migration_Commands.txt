# Entity Framework 6.x Migration Commands for Visual Studio Package Manager Console

## Step 1: Enable Migrations
Enable-Migrations -ContextTypeName Durdans_WebForms_MVP.Data.ClinicDbContext

## Step 2: Add Initial Migration
Add-Migration InitialCreate

## Step 3: Update Database (Create Schema)
Update-Database -Verbose

## Optional: View SQL that will be executed
Update-Database -Script

## If you need to rollback
Update-Database -TargetMigration:0

## To see migration history
Get-Migrations

## Step 4: Add User Table Migration (for Authentication)
Add-Migration AddUsersTable

## Step 5: Update Database with User Table
Update-Database -Verbose

## Step 6: Add User-Patient Linking (UserId to Patient Table)
Add-Migration AddUserIdToPatient

## Step 7: Update Database with User-Patient Link
Update-Database -Verbose
